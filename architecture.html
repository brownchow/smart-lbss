Smart LBSS 系统架构

智能锂离子电池储能管理系统 — 三层 IoT 架构

CLOUD LAYER
云端层
🖧
RCA
Python / Flask — 服务器
远程控制应用，轮询 uGrid 状态，存储遥测数据至 MySQL，通过 MQTT 发布告警，提供 REST API 供客户端调用。
Flask :3000 MySQL MQTT CoAP Client
💻
CA
Python / curses — 终端
客户端 TUI 仪表盘，实时显示电池状态、告警信息，支持手动控制（全放电、目标 SoC、MPC 调参）。
Curses TUI HTTP Client MQTT Sub
▼ CoAP GET /dev/state  |  HTTP REST API  |  MQTT ugrid/alerts/# ▲
AGGREGATION LAYER
聚合层
🔌
uGridController
nRF52840 — Contiki-NG
微电网中央控制器，管理多个电池，运行 ML 预测模型和 MPC 优化算法，每 5 秒下发充放电功率指令。
ML 推理 60→16→8→2 MPC 梯度下降 CoAP Server
📡
RPL Border Router
nRF52840 — Contiki-NG
边界路由器，将 6LoWPAN 低功耗无线传感器网络桥接到 IPv6 互联网，提供 RPL 路由和可选 Web 界面。
6LoWPAN RPL SLIP Tunnel
▼ CoAP Observe /dev/state  |  CoAP PUT /dev/power (milliwatts) ▲
EDGE LAYER
边缘层
🔋
BatteryController
nRF52840 — Contiki-NG
单电池控制器，运行嵌入式 ML 模型估算 SoH，执行安全检查（过温/低 SoH 自动隔离），通过 CoAP 上报状态并接收功率指令。
ML 推理 40→16→8→1 CoAP Server 安全隔离
⚡
Battery Pack
13.5kWh / 200Ah / 3.7V
锂离子电池组物理模型，模拟电压、电流、温度、SoC 的实时变化，包含内阻、转换效率、热模型和衰减模型。
物理仿真 热模型 SoH 衰减
数据流链路
Battery Pack → BatteryController
传感器数据（电压、电流、温度）每秒采样，SoC 通过库仑计数更新
BatteryController 内部处理
运行 battery_soh_regress ML 模型（~3.1KB），结合物理模型计算 SoH，执行安全检查
BatteryController → uGridController
通过 CoAP Observe 订阅 /dev/state，JSON 格式推送 V/I/T/SoC/SoH/State
uGridController 内部处理
运行 power_predictor_regress 预测 PV/负载，MPC 投影梯度下降（100 次迭代）计算最优功率分配
uGridController → BatteryController
每 5 秒 CoAP PUT /dev/power 下发 {"u": 毫瓦} 功率指令，控制充放电
uGridController → RCA
经 RPL Border Router 桥接，RCA 每 5 秒 CoAP GET /dev/state（CBOR 编码）获取全量状态
RCA → MySQL + MQTT + CA
遥测数据持久化至 MySQL，告警发布到 ugrid/alerts/#，REST API 供 CA 查询状态和下发控制指令
CA 用户操作 → RCA → uGridController → BatteryController
用户通过 TUI 发送目标 SoC/放电/调参指令，经 REST API → RCA → CoAP PUT /ctrl/obj → 功率指令下发
