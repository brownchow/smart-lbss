# 智能锂离子电池储能管理系统

本项目实现了一个**用于微电网和可再生能源场景下锂离子电池储能系统 (LBSS) 监控与控制的智能 IoT 系统**。  
该项目是比萨大学 *物联网* 课程的学术项目。

系统结合了**边缘计算、机器学习和预测控制**，以延长电池寿命、确保安全，并优化电池、可再生能源与主电网之间的能量流动。

---

## 🚀 核心功能

- **边缘级电池安全与健康监控**
- **基于机器学习的实时健康状态 (SoH) 估算**
- **基于模型预测控制 (MPC) 的预测性能量调度**
- **多层 IoT 架构（边缘层、聚合层、云端层）**
- **基于 CoAP over 6LoWPAN 的低功耗通信**
- **通过 MQTT 实现异步告警与监控**
- **通过 REST API 进行远程监控与控制**
- **故障隔离与安全优先设计**

---

## 🏗 系统架构

架构分为三个逻辑层：

### 🔹 边缘层
- **BatteryController（电池控制器）**
  - 运行于 nRF52840 设备
  - 监测电压、电流、温度、荷电状态 (SoC)
  - 执行设备端 ML 模型估算电池健康状态 (SoH)
  - 执行安全约束并控制功率继电器
  - 保证对关键故障的毫秒级响应

### 🔹 聚合层
- **uGridController（微电网控制器）**
  - 连接混合逆变器及多个 BatteryController
  - 使用 ML 预测负载和可再生能源发电
  - 通过 MPC 优化充放电决策
  - 作为电池遥测数据的代理
- **RPL Border Router（RPL 边界路由器）**
  - 将低功耗无线传感器网络桥接到 IPv6 互联网

### 🔹 云端层
- **RCA（远程控制应用）**
  - REST API（Flask）用于监控与控制
  - 通过 MySQL 进行数据持久化
  - 通过 MQTT 分发告警信息
- **CA（客户端应用）**
  - 基于 CLI 的实时监控界面
  - 订阅 MQTT 告警
  - 支持手动控制和 MPC 参数调节

## 🧠 机器学习与控制

### 电池健康估算
- ML 模型**直接在电池控制器上运行**
- 根据近期遥测数据估算健康状态 (SoH)
- 支持主动维护和故障隔离

### 预测性能量调度
- 负载和光伏发电在本地进行预测
- MPC 优化目标：
  - 电网能量交换成本
  - 电池衰减
  - 荷电状态 (SoC) 稳定性
- 使用**投影梯度下降法**求解优化问题，适用于嵌入式设备

## 🌐 通信协议栈

- **CoAP** 用于轻量级设备通信
- **6LoWPAN + IEEE 802.15.4** 用于低功耗组网
- **MQTT** 用于异步告警
- **HTTP/REST** 用于云端与客户端交互
- **CBOR 与 JSON** 用于高效载荷编码
