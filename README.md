# Smart Lithium-ion Battery Energy Storage Management

This project implements a **smart IoT system for monitoring and controlling Lithium-ion Battery Storage Systems (LBSS)** in microgrid and renewable energy scenarios.  
It was developed as an academic project for the *Internet of Things* course at the University of Pisa.

The system combines **edge computing, machine learning, and predictive control** to improve battery lifespan, ensure safety, and optimize energy flows between batteries, renewable sources, and the main grid.

---

## 🚀 Key Features

- **Edge-level battery safety and health monitoring**
- **Real-time State of Health (SoH) estimation using Machine Learning**
- **Predictive energy flow scheduling using Model Predictive Control (MPC)**
- **Multi-layer IoT architecture (Edge, Aggregation, Cloud)**
- **Low-power communication using CoAP over 6LoWPAN**
- **Asynchronous alerts and monitoring via MQTT**
- **Remote monitoring and control through REST APIs**
- **Fault isolation and safety-first design**

---

## 🏗 System Architecture

The architecture is organized into three logical layers:

### 🔹 Edge Layer
- **BatteryController**
  - Runs on nRF52840 devices
  - Monitors voltage, current, temperature, SoC
  - Executes an on-device ML model to estimate battery SoH
  - Enforces safety constraints and controls power relays
  - Guarantees millisecond-level reaction to critical faults

### 🔹 Aggregation Layer
- **uGridController**
  - Connected to a hybrid inverter and multiple BatteryControllers
  - Predicts load and renewable generation using ML
  - Optimizes charge/discharge decisions via MPC
  - Acts as a proxy for battery telemetry
- **RPL Border Router**
  - Bridges the low-power WSN to the IPv6 Internet

### 🔹 Cloud Layer
- **Remote Control Application (RCA)**
  - REST API (Flask) for monitoring and control
  - Data persistence via MySQL
  - Alert distribution via MQTT
- **Client Application**
  - CLI-based interface for real-time monitoring
  - Subscribes to MQTT alerts
  - Allows manual control and tuning of MPC parameters

## 🧠 Machine Learning & Control

### Battery Health Estimation
- ML model runs **directly on the battery controller**
- Estimates State of Health from recent telemetry
- Enables proactive maintenance and safety isolation

### Predictive Energy Scheduling
- Load and PV production are predicted locally
- MPC optimizes:
  - Grid energy exchange cost
  - Battery degradation
  - State of Charge stability
- Optimization solved with **Projected Gradient Descent**, suitable for embedded devices

## 🌐 Communication Stack

- **CoAP** for lightweight device communication
- **6LoWPAN + IEEE 802.15.4** for low-power networking
- **MQTT** for asynchronous alerts
- **HTTP/REST** for cloud-to-client interaction
- **CBOR & JSON** for efficient payload encoding
